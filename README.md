## A propos

Ce site est un projet Laravel 10 pour une remise à niveau, j'ai
utilisé :
* Laravel 10 (https://laravel.com/docs/10.x/releases)  
* VueJs (https://vuejs.org/)
* TailwindCss (https://tailwindcss.com/)
* Git (https://git-scm.com/downloads)
